# Webpack Template

Template repository for creating a new website via Webpack.

## Installation 

- You should have Node.js installed
- Clone this repo
- Open up VS Code (or your IDE of choice), and enter these two commands: 
  - npm install
  - npm start
- You're good to go! 

Note: If you're using TypeScript or simply want to enforce JS type safety, drag and replace the contents from the /typesafe folder into the root directory.
